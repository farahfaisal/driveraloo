# كيفية تثبيت التطبيق على الديسكتوب

## ⚠️ مهم: المتصفحات المدعومة

زر التثبيت التلقائي يعمل فقط على:
- ✅ **Chrome** (Windows/Mac/Linux)
- ✅ **Edge** (Windows/Mac/Linux)
- ❌ **Safari** (macOS) - غير مدعوم
- ❌ **Firefox** - غير مدعوم

## الطريقة الأولى: التثبيت التلقائي (Chrome/Edge)

1. افتح التطبيق في Chrome أو Edge
2. انتظر 3 ثواني
3. سيظهر زر التثبيت في أسفل الشاشة تلقائياً
4. اضغط "تثبيت"

## الطريقة الثانية: التثبيت اليدوي من شريط العنوان

### على Chrome:
1. ابحث عن أيقونة **⊕ التثبيت** أو **💻** في شريط العنوان (يمين الشاشة)
2. اضغط على الأيقونة
3. اضغط "تثبيت"

### على Edge:
1. اضغط على القائمة **⋯** (ثلاث نقاط)
2. اختر **"التطبيقات"** → **"تثبيت هذا الموقع كتطبيق"**
3. اضغط "تثبيت"

## الطريقة الثالثة: من صفحة الإعدادات

1. افتح التطبيق
2. اذهب إلى **الإعدادات** (آخر زر في القائمة السفلية)
3. ابحث عن قسم **"تثبيت التطبيق"**
4. اضغط على زر **"ثبت التطبيق الآن"**
5. أو اتبع التعليمات اليدوية الموجودة

## التشخيص: لماذا لا يظهر الزر؟

اذهب إلى **الإعدادات** → **تثبيت التطبيق** → **تشخيص PWA**

ستجد معلومات تفصيلية عن:
- حالة التثبيت الحالية
- دعم المتصفح
- Service Worker
- HTTPS
- وأكثر...

## حل المشاكل الشائعة

### المشكلة: الزر لا يظهر أبداً

**الحل:**
1. افتح Console في المتصفح (F12)
2. اكتب:
   ```javascript
   localStorage.removeItem('pwa-install-dismissed');
   location.reload();
   ```
3. انتظر 3 ثواني

### المشكلة: استخدم Safari على Mac

**الحل:**
Safari لا يدعم PWA بالطريقة التقليدية.
- جرّب **Chrome** أو **Edge** للحصول على أفضل تجربة
- أو: اضغط Share → Add to Dock (لكن بميزات محدودة)

### المشكلة: التطبيق مثبت بالفعل

**الحل:**
إذا كان التطبيق مثبتاً، لن يظهر زر التثبيت.
للتحقق:
```javascript
console.log(window.matchMedia('(display-mode: standalone)').matches);
// إذا كانت النتيجة true، التطبيق مثبت بالفعل
```

## مميزات PWA بعد التثبيت

- 🚀 فتح سريع من سطح المكتب
- 📱 يعمل مثل التطبيق العادي (بدون شريط المتصفح)
- 🔔 إشعارات فورية
- 📴 يعمل بدون إنترنت (بعض الميزات)
- 💾 لا يحتاج مساحة تخزين كبيرة

## للمطورين

### التحقق من PWA Readiness:

```javascript
// في Console
console.log('beforeinstallprompt:', 'onbeforeinstallprompt' in window);
console.log('Service Worker:', 'serviceWorker' in navigator);
console.log('Is Installed:', window.matchMedia('(display-mode: standalone)').matches);

// عرض Service Workers المسجلة
navigator.serviceWorker.getRegistrations().then(console.log);
```

### إجبار إعادة ظهور الزر:

```javascript
localStorage.removeItem('pwa-install-dismissed');
location.reload();
```

## ملاحظات

1. **HTTPS مطلوب**: PWA يعمل فقط على HTTPS أو localhost
2. **Service Worker**: يجب أن يكون مسجلاً بنجاح
3. **Manifest**: يجب أن يكون صحيحاً وموجوداً
4. **الأيقونات**: يجب أن تكون بالأحجام المناسبة

## الدعم

إذا استمرت المشكلة:
1. تحقق من Console للأخطاء
2. استخدم صفحة الإعدادات → تشخيص PWA
3. تأكد من استخدام Chrome أو Edge
4. أعد بناء المشروع: `npm run build`
