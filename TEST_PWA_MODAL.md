# اختبار النافذة المنبثقة لتثبيت PWA

## 🧪 خطوات الاختبار السريع

### الطريقة 1: في المتصفح مباشرة

1. افتح التطبيق في **Chrome** أو **Edge**
2. افتح Console (F12)
3. اكتب هذا الأمر:
   ```javascript
   localStorage.removeItem('pwa-install-dismissed');
   location.reload();
   ```
4. انتظر **3 ثواني**
5. ✅ يجب أن تظهر النافذة المنبثقة!

### الطريقة 2: من صفحة الإعدادات

1. افتح التطبيق
2. اذهب إلى **الإعدادات**
3. ابحث عن قسم **"تشخيص PWA"**
4. اضغط على زر **"إعادة تعيين حالة التثبيت"**
5. سيتم إعادة التحميل تلقائياً
6. ✅ ستظهر النافذة المنبثقة!

## 🎯 اختبار الأزرار

### اختبار زر "ثبت التطبيق الآن":
1. اضغط الزر الأزرق
2. ✅ يجب أن تظهر نافذة Chrome الأصلية
3. اضغط "Install"
4. ✅ يجب أن ترى رسالة نجاح ✓
5. ✅ النافذة تغلق تلقائياً بعد 2 ثانية

### اختبار زر "لاحقاً":
1. اضغط زر "لاحقاً" (الرمادي)
2. ✅ النافذة تغلق فوراً
3. أعد تحميل الصفحة
4. ✅ النافذة تظهر مجدداً!

### اختبار زر "لا، شكراً":
1. اضغط زر "لا، شكراً"
2. ✅ النافذة تغلق
3. أعد تحميل الصفحة
4. ❌ النافذة لا تظهر (محفوظ لمدة 3 أيام)
5. لإعادة إظهارها:
   ```javascript
   localStorage.removeItem('pwa-install-dismissed');
   location.reload();
   ```

## 🐛 حل المشاكل

### المشكلة: النافذة لا تظهر أبداً

**الأسباب المحتملة:**
1. ❌ تستخدم Safari أو Firefox
   - **الحل:** استخدم Chrome أو Edge
2. ❌ التطبيق مثبت بالفعل
   - **تحقق:** `window.matchMedia('(display-mode: standalone)').matches`
3. ❌ تم رفض التثبيت خلال آخر 3 أيام
   - **الحل:** `localStorage.removeItem('pwa-install-dismissed')`
4. ❌ الموقع ليس على HTTPS
   - **الحل:** استخدم localhost أو HTTPS

### المشكلة: النافذة تظهر ثم تختفي سريعاً

**الأسباب:**
- beforeinstallprompt لم يتم استلامه
- **الحل:** تحقق من Console للأخطاء

### المشكلة: زر "ثبت الآن" معطل (رمادي)

**الأسباب:**
- deferredPrompt غير متوفر
- **الحل:** أعد تحميل الصفحة

## 🔧 أوامر Console المفيدة

```javascript
// إعادة إظهار النافذة
localStorage.removeItem('pwa-install-dismissed');
location.reload();

// التحقق من حالة التثبيت
console.log('مثبت:', window.matchMedia('(display-mode: standalone)').matches);

// التحقق من الدعم
console.log('يدعم:', 'onbeforeinstallprompt' in window);

// التحقق من آخر رفض
const dismissed = localStorage.getItem('pwa-install-dismissed');
if (dismissed) {
  const days = Math.floor((Date.now() - parseInt(dismissed)) / (1000 * 60 * 60 * 24));
  console.log('تم الرفض قبل', days, 'يوم');
}

// محاكاة رفض قديم (أكثر من 3 أيام)
localStorage.setItem('pwa-install-dismissed', (Date.now() - 4 * 24 * 60 * 60 * 1000).toString());
```

## ✨ نصائح للتطوير

1. **اختبر في Chrome أولاً** - أسهل للتطوير
2. **استخدم Console** - للتحقق من الأخطاء
3. **امسح localStorage** - لإعادة الاختبار
4. **جرب جميع الأزرار** - تأكد من كل السيناريوهات
5. **اختبر على أجهزة مختلفة** - Desktop, Tablet, Mobile
