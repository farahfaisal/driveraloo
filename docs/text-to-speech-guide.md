# دليل الصوت الناطق (Text-to-Speech)

## نظرة عامة

تم إضافة ميزة **الصوت الناطق** (Text-to-Speech) للتطبيق لتحسين تجربة السائقين من خلال توفير إشعارات صوتية منطوقة باللغة العربية.

## الميزات الرئيسية

### 1. إشعار صوتي للرحلات المخصصة
عندما يتم تخصيص رحلة مباشرة للسائق:
- ✅ يظهر إشعار مرئي
- ✅ يتم تشغيل صوت تنبيه
- ✅ **يتم تشغيل صوت ناطق يقول: "لديك رحلة مخصصة - رقم الطلب [رقم]"**

### 2. إشعار صوتي للطلبات الجديدة
عندما يظهر طلب جديد في صفحة الطلبات:
- ✅ يظهر إشعار مرئي
- ✅ يتم تشغيل صوت تنبيه
- ✅ **يتم تشغيل صوت ناطق يقول: "طلب جديد - رقم الطلب [رقم]"**

## كيفية العمل

### التقنية المستخدمة
الميزة تستخدم **Web Speech API** المدمجة في المتصفحات الحديثة:
- لا تحتاج إلى مكتبات خارجية
- تعمل على جميع المتصفحات الحديثة (Chrome, Safari, Edge, Firefox)
- تدعم اللغة العربية بشكل كامل

### الملفات المضافة

#### 1. `src/utils/textToSpeech.ts`
ملف utility يحتوي على جميع وظائف TTS:

```typescript
// تشغيل صوت ناطق بنص مخصص
speak(text: string, options?: TTSOptions): Promise<void>

// إيقاف النطق
stopSpeaking(): void

// التحقق من حالة النطق
isSpeaking(): boolean

// الحصول على الأصوات المتاحة
getAvailableVoices(): Promise<SpeechSynthesisVoice[]>

// الحصول على الأصوات العربية فقط
getArabicVoices(): Promise<SpeechSynthesisVoice[]>

// وظائف مخصصة جاهزة
speakAssignedTrip(orderNumber?: string): Promise<void>
speakNewOrder(orderNumber?: string): Promise<void>
```

#### 2. التحديثات على `src/pages/MyTrips.tsx`
تم إضافة الصوت الناطق عند اكتشاف رحلة جديدة مخصصة:

```typescript
// تشغيل الصوت الناطق مع رقم الطلب
await speakAssignedTrip(orderNumber);
```

#### 3. التحديثات على `src/hooks/useOrderNotifications.ts`
تم إضافة الصوت الناطق عند اكتشاف طلب جديد:

```typescript
// تشغيل الصوت الناطق مع رقم الطلب
await speakNewOrder(orderNumber);
```

## خيارات التخصيص

يمكن تخصيص الصوت الناطق من خلال الخيارات:

```typescript
const options: TTSOptions = {
  volume: 1.0,      // مستوى الصوت (0-1)
  rate: 0.9,        // سرعة النطق (0.1-10)
  pitch: 1.1,       // طبقة الصوت (0-2)
  lang: 'ar-SA',    // اللغة (العربية السعودية)
  voice: 'صوت_محدد' // اسم صوت معين (اختياري)
};

await speak('لديك رحلة جديدة', options);
```

## سلوك الميزة

### السيناريوهات المختلفة

#### 1. رحلة مخصصة جديدة
```
📱 يظهر إشعار مرئي: "🚗 رحلة مخصصة!"
🔊 يتم تشغيل صوت تنبيه
🗣️ يتم نطق: "لديك رحلة مخصصة - رقم الطلب [رقم]"
```

#### 2. طلب جديد عادي
```
📱 يظهر إشعار مرئي: "🔔 طلب جديد!"
🔊 يتم تشغيل صوت تنبيه
🗣️ يتم نطق: "طلب جديد - رقم الطلب [رقم]"
```

#### 3. عدة طلبات متتالية
- يتم تشغيل إشعار وصوت ناطق لكل طلب
- تأخير ثانية واحدة بين كل طلب والآخر
- لا يتم قطع النطق السابق (تسلسل تلقائي)

### التعامل مع الأخطاء
- إذا لم يكن TTS مدعوماً: يتم تخطي النطق والاكتفاء بالصوت العادي
- إذا فشل النطق: يتم تسجيل الخطأ دون التأثير على الإشعارات الأخرى
- يتم طباعة رسائل واضحة في Console للتتبع

## اختبار الميزة

### في المتصفح (Development)
1. افتح صفحة الطلبات أو رحلاتي
2. أضف طلباً جديداً أو خصص رحلة
3. استمع للصوت الناطق

### في التطبيق (Android/iOS)
1. ابني التطبيق: `npm run android` أو `npm run ios`
2. افتح التطبيق على الجهاز
3. اختبر الإشعارات الصوتية

### اختبار يدوي في Console
```javascript
import { speakAssignedTrip, speakNewOrder } from './src/utils/textToSpeech';

// اختبار رحلة مخصصة
await speakAssignedTrip('1234');

// اختبار طلب جديد
await speakNewOrder('5678');
```

## الدعم والتوافق

### المتصفحات المدعومة
- ✅ Chrome/Edge (ممتاز - يدعم العربية بشكل كامل)
- ✅ Safari (جيد - يدعم العربية)
- ✅ Firefox (جيد - يدعم العربية)
- ⚠️ متصفحات قديمة: قد لا تدعم TTS

### أنظمة التشغيل
- ✅ Android: دعم كامل
- ✅ iOS: دعم كامل
- ✅ Windows: دعم ممتاز
- ✅ macOS: دعم ممتاز
- ✅ Linux: دعم جيد (يعتمد على النظام)

### الأصوات العربية
- يتم اختيار أفضل صوت عربي متاح تلقائياً
- إذا لم يتوفر صوت عربي، يتم استخدام الصوت الافتراضي
- يمكن للمستخدم تثبيت أصوات عربية إضافية من إعدادات النظام

## ملاحظات مهمة

### الأذونات
- ❗ بعض المتصفحات تتطلب تفاعل المستخدم أولاً قبل تشغيل الصوت
- ✅ في هذا التطبيق، يتم التعامل مع هذا تلقائياً

### الأداء
- ✅ TTS لا يستهلك bandwidth (يعمل locallly)
- ✅ لا يؤثر على سرعة التطبيق
- ✅ خفيف جداً على البطارية

### الخصوصية
- ✅ لا يتم إرسال أي بيانات إلى خوادم خارجية
- ✅ النطق يتم بالكامل على الجهاز
- ✅ لا يتم حفظ أي تسجيلات صوتية

## الإضافات المستقبلية الممكنة

### أفكار للتطوير
1. إضافة خيارات TTS في صفحة الإعدادات:
   - تفعيل/تعطيل الصوت الناطق
   - اختيار الصوت المفضل
   - ضبط السرعة والطبقة

2. نطق معلومات إضافية:
   - اسم العميل
   - عنوان التوصيل
   - قيمة الطلب

3. أوامر صوتية:
   - قبول الطلب بالصوت
   - بدء الرحلة بالصوت
   - إكمال التوصيل بالصوت

## الدعم الفني

إذا واجهت مشاكل مع الصوت الناطق:

3. جرب متصفحاً مختلفاً
4. تحقق من Console للحصول على رسائل الخطأ
