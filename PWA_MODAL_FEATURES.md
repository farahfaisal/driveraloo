# مميزات النافذة المنبثقة لتثبيت PWA

## 🎯 التوقيت والسلوك

### متى تظهر النافذة؟
- ✅ عند فتح التطبيق لأول مرة
- ✅ بعد **3 ثواني** من التحميل
- ✅ فقط على Chrome/Edge Desktop
- ✅ فقط إذا لم يكن التطبيق مثبتاً
- ✅ فقط إذا لم يرفض المستخدم خلال آخر 3 أيام

### متى لا تظهر؟
- ❌ إذا كان التطبيق مثبتاً بالفعل
- ❌ إذا رفض المستخدم خلال آخر 3 أيام
- ❌ على Safari أو Firefox
- ❌ على الموبايل Native App
- ❌ إذا لم يتم استلام `beforeinstallprompt`

## 🔘 الأزرار والخيارات

### 1️⃣ "ثبت التطبيق الآن" (أزرق)
```typescript
// عند الضغط:
- يعرض نافذة التثبيت الأصلية
- إذا قبل المستخدم:
  ✅ رسالة نجاح
  ✅ إغلاق تلقائي بعد 2 ثانية
  ✅ لن تظهر النافذة مجدداً
- إذا رفض:
  ❌ حفظ الرفض لمدة 3 أيام
```

### 2️⃣ "لاحقاً" (رمادي)
```typescript
// عند الضغط:
- ✅ إغلاق النافذة فقط
- ✅ لا يتم حفظ أي شيء
- ✅ ستظهر النافذة في الزيارة القادمة
```

### 3️⃣ "لا، شكراً" (خفيف)
```typescript
// عند الضغط:
- ❌ إغلاق النافذة
- ❌ حفظ الرفض في localStorage
- ❌ لن تظهر لمدة 3 أيام
- ✅ بعد 3 أيام، ستظهر مجدداً
```

### ✕ زر الإغلاق (أعلى اليسار)
```typescript
// نفس تأثير "لا، شكراً":
- ❌ إغلاق وحفظ لمدة 3 أيام
```

## 🎬 الأنيميشن

### عند الظهور:
1. **fadeIn** (0.3s): الخلفية السوداء الشفافة تظهر تدريجياً
2. **slideUp** (0.4s): النافذة تنزلق من الأسفل مع تكبير تدريجي

### التأثيرات التفاعلية:
- **hover**: تكبير الأزرار + تغيير اللون
- **active**: تصغير طفيف عند الضغط (scale-95)
- **disabled**: تحويل إلى رمادي مع cursor-not-allowed

## 💾 التخزين المحلي

### مفتاح localStorage:
```typescript
key: 'pwa-install-dismissed'
value: timestamp (رقم - Date.now())
```

### متى يتم الحفظ؟
- عند اختيار "لا، شكراً"
- عند الضغط على ✕
- عند رفض نافذة التثبيت الأصلية

### متى يتم المسح؟
- عند قبول التثبيت بنجاح
- يدوياً من Console

## 🧪 اختبار النافذة

### إعادة إظهار النافذة:
```javascript
// في Console
localStorage.removeItem('pwa-install-dismissed');
location.reload();
```

### التحقق من حالة الرفض:
```javascript
const dismissed = localStorage.getItem('pwa-install-dismissed');
if (dismissed) {
  const date = new Date(parseInt(dismissed));
  const days = (Date.now() - parseInt(dismissed)) / (1000 * 60 * 60 * 24);
  console.log('تم الرفض في:', date);
  console.log('منذ:', Math.floor(days), 'يوم');
}
```

### التحقق من حالة التثبيت:
```javascript
const isInstalled = window.matchMedia('(display-mode: standalone)').matches;
console.log('مثبت؟', isInstalled ? 'نعم ✅' : 'لا ❌');
```

## 🎯 حالات الاستخدام

### السيناريو 1: مستخدم جديد على Desktop
```
1. يفتح التطبيق في Chrome
2. ينتظر 3 ثواني
3. ✅ تظهر النافذة المنبثقة
4. يضغط "ثبت التطبيق الآن"
5. ✅ يتم التثبيت
6. ✅ رسالة نجاح
7. ✅ لن تظهر النافذة مجدداً
```

### السيناريو 2: مستخدم لا يريد التثبيت الآن
```
1. يفتح التطبيق
2. تظهر النافذة
3. يضغط "لاحقاً"
4. ✅ تغلق النافذة
5. في الزيارة القادمة:
   → تظهر النافذة مجدداً
```

### السيناريو 3: مستخدم لا يريد التثبيت أبداً
```
1. يفتح التطبيق
2. تظهر النافذة
3. يضغط "لا، شكراً" أو ✕
4. ✅ تغلق وتحفظ لمدة 3 أيام
5. خلال 3 أيام:
   → لن تظهر النافذة
6. بعد 3 أيام:
   → تظهر النافذة مجدداً
```

### السيناريو 4: التطبيق مثبت بالفعل
```
1. يفتح التطبيق
2. ❌ لا تظهر النافذة
3. السبب: التطبيق مثبت بالفعل
```

## 🔍 التشخيص

### في Console، ابحث عن:
```
PWA Modal: تهيئة النافذة المنبثقة
PWA Modal: المتصفح: Chrome/...
PWA Modal: يدعم beforeinstallprompt: true
PWA: ✅ تم استلام beforeinstallprompt event
```

### إذا لم تظهر النافذة:
```javascript
// 1. تحقق من الدعم
console.log('1. يدعم beforeinstallprompt:', 'onbeforeinstallprompt' in window);

// 2. تحقق من التثبيت
console.log('2. مثبت:', window.matchMedia('(display-mode: standalone)').matches);

// 3. تحقق من الرفض
const dismissed = localStorage.getItem('pwa-install-dismissed');
if (dismissed) {
  const days = (Date.now() - parseInt(dismissed)) / (1000 * 60 * 60 * 24);
  console.log('3. تم الرفض قبل:', Math.floor(days), 'يوم');
} else {
  console.log('3. لم يتم الرفض مسبقاً');
}

// 4. تحقق من Service Worker
navigator.serviceWorker.getRegistrations().then(regs => {
  console.log('4. Service Workers:', regs.length);
});
```

## 📝 الملاحظات الهامة

1. **النافذة تظهر مرة واحدة فقط لكل زيارة**
2. **لا تظهر على الموبايل Native App**
3. **تحتاج HTTPS (أو localhost)**
4. **تحتاج Chrome أو Edge Desktop**
5. **لا تعمل على Safari أو Firefox**
6. **الانتظار 3 ثواني يمنع الإزعاج**
7. **3 أيام cooldown لمن يرفضون**
