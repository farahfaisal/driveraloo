## 📱 اختبار على الجهاز

### 1. الفتح الأول للتطبيق
```
□ شاشة التحميل تظهر
□ نافذة "السماح بالإشعارات" تظهر تلقائياً
□ بعد القبول: لا إشعارات وهمية
□ نافذة "السماح بالموقع" تظهر تلقائياً
□ بعد القبول: لا إشعارات وهمية
□ التطبيق يفتح بدون إشعارات
```

### 2. اختبار الإشعارات
```
□ لا إشعارات عند فتح التطبيق
□ لا إشعارات للطلبات القديمة
□ إشعار يظهر عند وصول طلب جديد فقط
□ صوت الإشعار يعمل
□ الضغط على الإشعار يفتح صفحة الطلبات
```

### 3. اختبار الموقع
```
□ الموقع يعمل على صفحة الخريطة
□ تتبع GPS يعمل بشكل صحيح
□ لا أخطاء في الأذونات
```

## 🔧 إذا فشل الاختبار

### إذا لم تظهر نافذة الأذونات:
```bash
# 1. تحقق من المزامنة
npx cap sync android

# 2. أعد بناء APK
cd android
./gradlew clean
./gradlew assembleDebug

# 3. أعد تثبيت التطبيق
adb uninstall com.tayardriver.app
adb install app/build/outputs/apk/debug/app-debug.apk
```

### إذا ظهرت إشعارات وهمية:
```bash
# 1. تحقق من أن الملفات محدثة
grep -r "useOrderNotifications" src/pages/

# يجب أن يكون الناتج فارغاً أو فقط في hook file

# 2. أعد البناء والمزامنة
npm run build
npx cap sync android
```

### إذا لم تعمل السجلات:
```bash
# تحقق من اتصال الجهاز
adb devices

# امسح السجلات القديمة
adb logcat -c

# ابدأ عرض السجلات
adb logcat | grep "tayardriver"
```
